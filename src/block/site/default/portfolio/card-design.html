Квартира для семейной пары

Двухкомнатная квартира в центре города для семейной пары, которая ценит комфорт, уют и функциональность. При разработке дизайн-проекта стояла задача создать помещение, отвечающее всем этим критериям.

В качестве базовых для отделки помещений использовались пастельные цвета, мебель подбиралась монохромная, но итоговый вариант не лишен яркости и акцентов — за счет аксессуаров и декора.

Санузел

Ванная и туалет облицованы крупноформатной плиткой, по дизайну имитирующей натуральный мрамор. При этом для выделения зон использовались два ее типа — гладкая и с волнообразной фактурой. Ванна и раковина расположены вплотную, но разграничены перегородкой из стекла. Для каждой зоны предусмотрено свое освещение, есть подсветка зеркала, которая, попадая на фактурную плитку, создает в помещении особую камерную атмосферу.

В санузле из обычного короба, за которым спрятана разводка труб, мы сделали дизайнерскую инсталляцию. Достичь этого получилось за счет двух уровней конструкции, подсветки и декора фактурной плиткой.

Спальная комната

В спальне помимо окна есть выход на лоджию: это хорошо для освещения, но создает трудности при планировании интерьера. Пространство комнаты мы организовали вокруг кровати. Ее большое изголовье визуально продолжает стеклянное панно с абстрактным принтом.

Чтобы у хозяев была большая система хранения, пришлось сделать перепланировку и перенести дверной проем — в итоге встроенный гардероб с раздвижными дверями полностью занял одну из стен.

Идеальная для отдыха гостиная

При проектировании гостиной перед нами стояла задача создать комнату идеальную для отдыха. Мебель подбиралась располагающая к покою. Огромный диван так и приглашает прилечь. Белая корпусная мебель подвешена на стене, за счет чего создается ощущение ее невесомости. Этот эффект усиливается с помощью встроенной подсветки полок и декоративной стеклянной конструкции снизу. Спокойный, прозрачный оттенок стен поддержан более насыщенными цветами текстиля.
